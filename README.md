# Volunteer Coordination Toolkit

A one-page, portfolio-safe case study for Iliana Garcia's Volunteer Coordinator application package.

## Files required

- index.html
- styles.css
- assets/tails-around-town-flyer.png
- assets/volunteer-recognition-spotlight.png
- assets/volunteer-interest-open-house.png
- assets/paws-connect-newsletter.png
- assets/core-competencies-comic.png

Upload all files exactly as shown. The images must stay inside the assets folder.

## GitHub Pages setup

1. Create a public repository named `mcacc-volunteer-coordinator-portfolio`.
2. Upload `index.html`, `styles.css`, and `README.md` to the root of the repository.
3. Create an `assets` folder and upload the images into that folder.
4. Go to Settings > Pages.
5. Set Source to `Deploy from a branch`.
6. Set Branch to `main` and Folder to `/root`.
7. Save and wait for GitHub Pages to publish.
